## 상세 시나리오 결과

### S1: 회원가입 → 로그인

**목적:** Supabase Auth 기반 회원가입 및 로그인 플로우 전체 검증

**테스트 단계:**

1. Supabase Auth Admin API로 테스트 사용자 생성 (user@test.com)
2. 로그인 페이지(`/auth/login`) 접속 → 이메일/비밀번호 폼, 카카오 OAuth 버튼, 회원가입 링크 확인
3. 이메일/비밀번호 입력 → 로그인 성공 → 메인 페이지 리다이렉트
4. 네비바에 "마이페이지" 링크 + 유저 아바타 표시 확인

**결과: PASS**

---

### S2: 상품 필터링 + 상세 탭 전환

**목적:** 상품 목록 필터링 및 상세 페이지 탭 UI 정상 동작 검증

**테스트 단계:**

1. `/products` 접속 → 4개 상품 카드 로드 확인
2. "1인용" 필터 클릭 → 베이직 솔로 + 라이트 솔로 (2개) 표시
3. "2인용" 필터 클릭 → 프리미엄 듀오 + 풀패키지 (2개) 표시
4. `/products/1` 접속 → 브레드크럼, 이미지, 가격(₩59,000 / ₩79,000), 서비스 안내 확인
5. "포함 장비" 탭 클릭 → 6개 장비 아이템 표시 (돔텐트, 침낭, 에어매트, 버너, 랜턴, 가방)
6. "이용 안내" 탭 클릭 → 배송/반납/검수/취소 정책 텍스트 확인
7. "후기(0)" 탭 표시 확인

**결과: PASS**

### S3: 예약 전체 플로우 (인증 사용자)

**목적:** 4단계 예약 위자드 전체 플로우 및 가격 합산 정확성 검증

**테스트 단계:**

1. 로그인 상태에서 `/booking/1` 접속

2. **Step 1 — 렌탈 타입 선택**
   - 1박 2일 카드 선택
   - 올리브 보더 활성화 확인
   - "추천" 태그, ₩59,000 표시 확인
   - "다음 단계" 버튼 활성화 확인

3. **Step 2 — 날짜 선택**
   - 4월 캘린더 로드 확인
   - 4/1 비활성화 확인 (minBookableDate 이전)
   - 4/18(토) 선택 → 종료일 4/19(일) 자동 계산 확인
   - "잔여 세트: 10세트 남음" 표시 확인

4. **Step 3 — 소모품 옵션 선택**
   - 4개 소모품 표시 확인: 장작 ₩8,000, 숯 ₩6,000, 모기향 ₩3,000, 부탄가스 ₩5,000
   - 장작 수량 +1 클릭 → 수량 "1" 표시 확인

5. **Step 4 — 주문 요약**
   - 상품명: 베이직 솔로 세트
   - 렌탈 타입: 1박 2일
   - 이용 기간: 4.18.(토) ~ 4.19.(일)
   - 선택 옵션: 장작 1단 x1 → ₩8,000
   - 기본 렌탈료: ₩59,000
   - 옵션 합계: ₩8,000
   - **총 결제 금액: ₩67,000** (정확히 합산됨)
   - 사이드바 스텝 인디케이터: 완료 단계 체크마크, 현재 단계 올리브 하이라이트 확인

**결과: PASS**
